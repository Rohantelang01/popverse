## ğŸ—„ï¸ Database Design

### Collection 1: Users
```javascript
{
  _id: ObjectId,
  name: String,              // "Rahul Kumar"
  email: String,             // "rahul@example.com" (unique, lowercase)
  password: String,          // Hashed with bcrypt
  role: String,              // "user" | "admin"
  createdAt: Date,
  updatedAt: Date
}
```
**Indexes:** `email` (unique)

---

### Collection 2: Categories
```javascript
{
  _id: ObjectId,
  name: String,              // "Movies"
  slug: String,              // "movies" (unique)
  description: String,       // "Latest movie news and reviews..."
  createdAt: Date
}
```
**Pre-populated Data:**
- Movies, Webseries, Anime, Comics, Mangas, Novels, Games

**Indexes:** `slug` (unique)

---

### Collection 3: Tags
```javascript
{
  _id: ObjectId,
  name: String,              // "Action"
  slug: String,              // "action" (unique)
  usageCount: Number,        // Track popularity
  createdAt: Date
}
```
**Indexes:** `slug` (unique), `usageCount` (descending)

---

### Collection 4: Articles (Most Important)
```javascript
{
  _id: ObjectId,
  slug: String,              // "avatar-way-of-water-review" (unique)
  
  // English Content
  title_en: String,
  summary_en: String,
  content_en: String,        // Full MDX/HTML content
  
  // Hinglish Content
  title_hi: String,
  summary_hi: String,
  content_hi: String,
  
  // Media
  coverImage: {
    url: String,             // Cloudinary URL
    alt: String
  },
  
  // Classification
  categoryId: ObjectId,      // Single category (ref: Categories)
  tags: [ObjectId],          // Multiple tags (ref: Tags)
  
  // Author
  authorId: ObjectId,        // ref: Users
  
  // Status & Publishing
  status: String,            // "draft" | "published" | "scheduled"
  publishedAt: Date,
  scheduledFor: Date,        // For scheduled posts
  
  // Engagement
  views: Number,             // Total views (increments on each visit)
  likes: Number,             // Total likes
  
  // SEO
  metaDescription_en: String,
  metaDescription_hi: String,
  metaKeywords: [String],
  
  // Review-specific (Optional)
  isReview: Boolean,
  rating: Number,            // 1-5 stars
  pros: [String],
  cons: [String],
  
  // Timestamps
  createdAt: Date,
  updatedAt: Date
}
```
**Indexes:** 
- `slug` (unique)
- `status` + `publishedAt` (compound)
- `categoryId` + `publishedAt`
- `views` (descending)
- `scheduledFor`

---

### Collection 5: Comments
```javascript
{
  _id: ObjectId,
  articleId: ObjectId,       // ref: Articles
  userId: ObjectId,          // ref: Users (login required)
  content: String,
  
  // Nested Replies
  parentId: ObjectId | null, // null = top-level, else parent comment
  
  // Moderation
  isFlagged: Boolean,        // User reported
  isDeleted: Boolean,        // Soft delete
  
  createdAt: Date,
  updatedAt: Date
}
```
**Indexes:**
- `articleId` + `createdAt`
- `parentId`
- `isFlagged`

**Comment Structure Example:**
```
Comment 1 (parentId: null)
  â”œâ”€â”€ Reply 1.1 (parentId: Comment1_ID)
  â””â”€â”€ Reply 1.2 (parentId: Comment1_ID)
```

---

### Collection 6: AdSlots
```javascript
{
  _id: ObjectId,
  name: String,              // "Header Banner"
  position: String,          // "header" | "sidebar" | "inline" | "footer"
  
  // Flexible Content
  type: String,              // "html" | "image" | "script"
  content: String,           // HTML/script code
  
  // Image type
  imageUrl: String,
  clickUrl: String,
  
  isActive: Boolean,
  createdAt: Date,
  updatedAt: Date
}
```
**Indexes:** `position` + `isActive`
